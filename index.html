about me

hello! i'm jasper. my pronouns are xie/xem/xyr, they/them, and it/its.
i make code, art, and languages, sometimes.
everything i code is FOSS. i usually license under GNU licenses (i.e., AGPL, GPL, LGPL). what i code is Linux-first. Windows will be accounted, for, just... you know...
i code in Python (and a bit of JavaScript), and i plan to learn systems programming languages, like C and Rust. i make games, libraries, tools, websites, and whatever i feel like making.
i also really like linguistics and conlanging (neography is fun too). i'm currently learning German. i have yet to publish a conlang, but i'll do that some day.
i mainly make pixel art (since it's most of what i'm good at), though i occasionally make a vector graphic or two (usually for something, like a logo or UI element), or a flag.
also, i'm non-binary, omnisexual, and ambiamorous :-) (happy pride month, woohoo)
i use arch btw :-3
